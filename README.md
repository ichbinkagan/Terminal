# Mini Terminal (Python)

I created a basic terminal in Python.  
It supports simple commands like `cd`, `ls`, `pwd`, `mkdir`, `clear`, and `exit`.

However, `mkdir` doesn’t work properly in some cases (still debugging it).  

You can modify, improve, or use this project however you want :)

## Features
- `cd <folder>` → change directory  
- `pwd` → show current directory  
- `ls` → list files and folders  
- `mkdir <folder>` → create a folder (partially working / under improvement)  
- `clear` → clear terminal  
- `exit` → close terminal  

## Notes
- This is a beginner-level project.
- Some commands may not handle all edge cases yet.
- Contributions and improvements are welcome.
